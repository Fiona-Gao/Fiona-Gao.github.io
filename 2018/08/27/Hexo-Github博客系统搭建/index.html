Hexo+Github博客系统搭建

Mac下Hexo+GitHub搭建博客（未完成，图片未插入）
一、 环境配置
1.Git
2.Node.js
3.更新node和npm
sudo n stable
4.Github账号
二、安装Hero
sudo npm install -g hexo
安装成功会出现一个非常长的树结构图

npm install
在用户名目下创建hexo文件夹，进入文件夹执行以下命令初始化：
hexo init可能会出现一些warning，先不管他。
hexo命令：
hexo generate / 生成一套静态网页 /
hexo server / 在服务器上运行 /
在http://localhost:4000可以看到以下界面：
git部署
安装git部署器
npm install hexo-deployer-git —save

这个是说有一个东西版本太低？先跳过。
创建一个Github仓库fiona-gao.github.io
复制链接：https://github.com/Fiona-Gao/Fiona-Gao.github.io
修改_config.yml文件，进入hexo，修改文件：
会看到很多属性（参考https://blog.csdn.net/jasonjwl/article/details/52887575）
配置完成后执行：hexo d
部署成功，可以在浏览器中查看结果：

当新增文章插件时可以执行以下步骤：
hexo clean
hexo generate
hexo deploy

使用：

hexo new post “我的第一篇博客”，创建一个post，会创建一个.md
修改这个md
然后执行
hexo generate
hexo deploy
hero server
